## 学习总览

本计划基于课程的 200 + 课时，按 **“前端→工程化→后端→数据库→数据访问”** 的逻辑顺序编排，建议每周学习 5 天，每天投入 2-3 小时，周末用于复习和项目练习，整体周期约 8-10 周。

# 第一阶段：前端基础与进阶（第1-3周）

## 第 1 周：Web 前端基础（HTML+CSS）

### 第 1 天：理解Web开发流程，掌握HTML/CSS基础语法。

任务：1.安装Chrome开发者工具并熟悉Elements面板；2.编写第一个静态页面，包含<h1>标题、<p>段落、<img>图片标签；3.通过内联样式修改文字颜色和图片大小。

### 第 2 天：熟练使用VsCode开发工具，掌握文本/列表/链接标签及基础CSS选择器。

任务：1. 配置VsCode自动保存、代码格式化插件；2. 用<ul>/<ol>实现“新闻分类列表”；3. 用类选择器（.class）和标签选择器统一列表样式（字体、间距）；4. 完成“仿央视新闻头部导航”静态布局。

### 第 3 天：掌握CSS盒模型、浮动布局，解决浮动塌陷问题。

任务：1. 用盒模型（width/height/padding/margin）实现“新闻卡片”模块（含边框、内边距）；2. 用float实现“左图右文”的新闻列表布局；3. 通过“伪元素清除浮动”解决布局错乱；4. 优化央视案例的响应式适配（缩小窗口不重叠）。

### 第 4 天：掌握CSS定位、背景样式，实现复杂模块排版。

任务：1. 用position:relative/absolute实现“新闻热点标签”（右上角悬浮）；2. 用background-image设置页面背景图并调整透明度；3. 实现“新闻正文页”的三栏布局（左导航+中内容+右推荐）；4. 统一页面字体和颜色风格，提升美观度。

### 第 5 天：掌握表单标签及样式美化，完成综合案例。

任务：1. 用<form><input><button><select>等标签实现“用户登录页”；2. 美化表单（输入框边框、按钮 hover 效果、表单居中）；3. 给表单添加基础校验（必填项提示）；4. 提交代码到本地Git仓库（初始化仓库+提交记录）。


## 第 2 周：Web 前端进阶（CSS 深化 + JS 基础）

### 第 1 天（周一）：掌握 CSS 响应式布局、表单高级样式

任务：

1. 用媒体查询（@media）优化登录页，适配手机端（输入框宽度 100%）；
2. 实现 “用户注册页”（含用户名、密码、手机号、性别选择）；
3. 给表单添加焦点样式（输入框选中时边框变色）；
4. 测试不同设备尺寸下的页面显示效果。

### 第 2 天（周二）：复盘 HTML/CSS 核心知识点，入门 JavaScript

任务：

1. 整理 HTML 常用标签（文本 / 表单 / 容器）和 CSS 核心技能（选择器 / 盒模型 / 布局）思维导图；
2. 掌握 JS 的 3 种引入方式（内联 / 内部 / 外部）；
3. 编写外部 JS 文件，实现 “控制台打印 Hello JS” 和 “页面加载后弹出欢迎提示”。

### 第 3 天（周三）：掌握 JS 变量声明、数据类型及函数定义

任务：

1. 练习 let/const/var 的区别（变量提升、作用域测试）；
2. 实现 “数据类型转换” 函数（字符串转数字、布尔值判断）；
3. 编写 “计算两个数之和”“判断奇偶数” 的函数；
4. 用 console.log 调试函数返回值，验证正确性。

### 第 4 天（周四）：掌握 JS 对象、JSON 语法及 DOM 基础操作

任务：

1. 定义 “用户对象”（含姓名、年龄、性别属性，及打招呼方法）；
2. 实现对象与 JSON 的相互转换（JSON.stringify/JSON.parse）；
3. 用 DOM 操作获取登录页的输入框值，修改页面标题和按钮文字；
4. 实现 “点击按钮隐藏 / 显示表单” 功能。

### 第 5 天（周五）：掌握 JS 事件监听，完成交互功能开发

任务：

1. 给注册页添加事件：输入框失焦时校验手机号格式（正则匹配）；
2. 实现 “密码可见 / 隐藏” 切换（点击眼睛图标修改 input 的 type 属性）；
3. 实现 “表单提交前校验”（必填项为空时阻止提交并提示）；
4. 优化交互体验（校验失败时输入框边框标红）。



## 第 3 周：前端框架（Vue）+ 异步请求（Ajax）

### 第 1 天（周一）：掌握 Vue 实例创建、插值表达式及 v-for 列表渲染

学习课时：25.Vue 快速入门、26.v-for 指令任务：

1. 安装 Vue（CDN 引入），创建第一个 Vue 实例；
2. 用插值表达式（`{{}}`）渲染页面数据；
3. 用`v-for`渲染 “待办事项列表”（数组数据）；
4. 实现 “添加待办项” 功能（`push`数组元素，页面自动更新）。

### 第 2 天（周二）：掌握 Vue 属性绑定、条件渲染

学习课时：27.v-bind&v-if&v-show任务：

1. 用`v-bind`动态绑定图片`src`、按钮`disabled`属性；
2. 用`v-if/v-else`实现 “登录 / 未登录状态切换”（根据`isLogin`变量显示不同按钮）；
3. 用`v-show`实现 “待办项完成状态切换”（完成后隐藏编辑按钮）；
4. 结合`v-for`和`v-bind`给每个待办项添加唯一`class`。

### 第 3 天（周三）：掌握 Vue 双向数据绑定和事件绑定

学习课时：28.v-model 与 v-on任务：

1. 用`v-model`实现 “待办项输入框” 与数据的双向绑定；
2. 用`v-on`绑定点击事件，实现 “添加待办”“删除待办” 功能；
3. 实现 “待办项数量统计”（实时显示已完成 / 总数量）；
4. 优化事件处理（传递参数区分不同待办项）。

### 第 4 天（周四）：理解异步请求原理，掌握 Ajax 基础用法

学习课时：29.Ajax 入门、30.Ajax 案例任务：

1. 理解 “同步 vs 异步” 区别，用`XMLHttpRequest`实现第一个 Ajax 请求；
2. 调用 “天气 API”（如聚合数据免费接口），获取并打印城市天气数据；
3. 实现 “输入城市名查询天气” 功能（点击按钮发起请求，渲染结果到页面）；
4. 处理请求异常（网络错误时显示提示）。

### 第 5 天（周五）：整合 HTML/CSS/JS/Vue/Ajax 知识，完成综合项目

学习课时：前端模块总结 + 静态项目实战任务：

1. 开发 “个人博客静态站”，包含 3 个页面：首页（文章列表）、详情页（文章内容）、留言板（提交留言）；
2. 首页用 Vue 渲染文章列表，详情页用 Ajax 模拟请求文章内容；
3. 留言板用`v-model`收集留言信息，提交后用 Ajax 模拟保存并刷新列表；
4. 统一页面样式，实现页面跳转（用`a`标签或 Vue 路由基础）。


# 第二阶段：工程化与后端基础（第4-8周）

## 第 4 周：工程化工具（Maven）+ 单元测试

### 第 1 天（周一）：理解 Maven 核心作用，完成环境搭建

学习课时：31.Maven 课程介绍、32. 概述 & 安装、33.IDEA 集成任务：

4. 创建第一个 Maven Java 项目（jdk1.8）。

### 第 2 天（周二）：掌握 Maven 依赖配置，入门单元测试

学习课时：34. 依赖管理、35. 单元测试概述 & 入门任务：

1. 理解`pom.xml`结构，在`dependencies`中引入 JUnit 4 依赖；
2. 编写 “计算工具类”（含加法、减法、乘法方法）；
3. 编写单元测试类，用`@Test`注解测试每个方法；
4. 运行测试用例，查看测试结果（成功 / 失败原因）。

### 第 3 天（周三）：掌握单元测试断言和注解，规范代码编写

学习课时：36. 断言 & 常见注解、37. 企业开发规范 & AI 生成任务：

1. 用`Assert`断言（`assertEquals`、`assertTrue`）优化测试用例；
2. 熟悉`@Before`（测试前初始化）、`@After`（测试后清理）注解；
3. 按企业规范命名类和方法（如`CalculationUtil`、`add(int a, int b)`）；
4. 尝试用 AI 生成测试用例，对比手动编写的差异并优化。

### 第 4 天（周四）：理解 Maven 依赖范围，解决常见问题

学习课时：38.Maven 依赖范围、39. 常见问题解决任务：

1. 测试`compile`（编译运行都有效）、`test`（仅测试有效）、`provided`（编译有效运行无效）的区别；
2. 解决 “依赖冲突”（用`mvn dependency:tree`查看依赖树，排除冲突依赖）；
3. 解决 “依赖下载慢”（确认阿里云镜像配置正确）；
4. 练习 Maven 命令（`mvn clean`、`mvn compile`、`mvn test`）。

### 第 5 天（周五）：整合 Maven 和单元测试，完成工具类项目

学习课时：Maven + 单元测试实战任务：

1. 开发 “字符串工具类”（含判空、反转、去重方法）；
2. 用 Maven 管理依赖（仅引入必要依赖）；
3. 编写完整单元测试用例（覆盖正常场景和异常场景，如传入`null`值）；
4. 运行`mvn package`打包项目，查看`target`目录下的 jar 包；
5. 提交项目到 Git（含`pom.xml`、源码和测试类）。

### 第 5 天（周五）：理解三层架构和 IOC/DI，优化项目结构

学习课时：47. 分层解耦 - 三层架构、48.IOC 与 DI 入门任务：

## 第 6 周：后端进阶（IOC/DI+MySQL 基础）

学习课时：50.MySQL 课程介绍、51. 安装 & 数据模型任务：

学习课时：52.DDL - 数据库操作、53.DDL - 创建表任务：

   ```sql

       username VARCHAR(20) NOT NULL,
       password VARCHAR(32) NOT NULL,
   ```

学习课时：56.DDL - 查询表结构、57.DML-insert&update任务：

学习课时：63.JDBC - 执行 DQL 语句任务：
